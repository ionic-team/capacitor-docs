# Capacitor CLI - cap add

将原生平台项目添加至您的应用。

```bash
npx cap add <platform>
```

<strong>输入参数：</strong>

- `platform`（必填）: `android`, `ios`

<strong>选项：</strong>

仅限 iOS：
- `--packagemanager`: `SPM`, `Cocoapods`
