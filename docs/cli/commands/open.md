# Capacitor CLI - cap open

在指定的原生 IDE（iOS 使用 Xcode，Android 使用 Android Studio）中打开原生项目工作区。

```bash
npx cap open <platform>
```

<strong>输入参数：</strong>

- `platform`（必需）：`android`、`ios`
