# Capacitor CLI

Capacitor 命令行界面（CLI）工具用于开发 Capacitor 应用。

## CLI 参考

```bash
npx cap [-V] [-h] [<command>]
```

<strong>选项：</strong>

- `-V, --version`（可选）：输出版本号
- `-h, --help`（可选）：输出使用信息。也可用于单个命令。
