# Capacitor 插件

在 Capacitor 中，插件让 JavaScript 能够直接与原生 API 交互。

<br />

通过插件，Web 应用可以充分利用原生 API 的强大功能。插件封装了常见的原生操作，这些操作在不同平台上可能使用差异较大的 API，而插件则为 JavaScript 提供了一致的跨平台 API。

此外，Capacitor 的插件功能使得传统原生开发者和 Web 开发者可以协同工作，分别负责应用的不同部分。

Capacitor 会在客户端自动生成 JavaScript 钩子，因此大多数插件只需在 iOS 上使用 Swift/Obj-C，在 Android 上使用 Java/Kotlin。当然，为插件添加自定义 JavaScript 也是可行的。
