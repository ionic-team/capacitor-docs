# 构建 Ionic 照片墙应用

**支持平台**：网页、iOS、Android

Capacitor 让构建能够在 iOS、Android、桌面端和网页端原生运行的 Web 应用变得简单。在本指南中，我们将构建一个可在所有平台上运行的完整照片墙应用。

> 本指南现由 Ionic 文档团队维护。最新版本包含针对 [Angular、React 和 Vue 的教程](https://ionicframework.com/docs/intro/next)。
