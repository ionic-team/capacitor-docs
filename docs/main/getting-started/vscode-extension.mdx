[WebNative Visual Studio Code 扩展](https://marketplace.visualstudio.com/items?itemName=webnative.webnative)是一个由社区维护的插件，可帮助您执行常见的 Capacitor 开发任务，无需记住 CLI 命令。

如果您的计算机上已安装 VS Code，请点击下方的安装按钮。您也可以通过搜索 "WebNative" 来找到此扩展。

<h2>附加文档</h2>

WebNative 扩展的完整文档可在 [webnative.dev](https://webnative.dev/introduction/getting-started/) 找到，涵盖以下主题：
- 构建、调试和运行您的 Capacitor 应用程序
- 包分析和依赖升级
- 从 Cordova 迁移
- 更改原生设置
- 启动画面和图标
- 使用 WebNative 应用在非 Mac 设备上进行开发
