# Capacitor CLI - cap open

在指定的原生 IDE 中打开原生项目工作区(iOS 为 Xcode，Android 为 Android Studio)。

```bash
npx cap open <platform>
```

<strong>输入:</strong>

- `platform` (必需): `android`、`ios`
