# Capacitor CLI - cap add

向您的应用添加一个原生平台项目。

```bash
npx cap add <platform>
```

<strong>输入:</strong>

- `platform` (必需): `android`、`ios`

<strong>选项:</strong>

仅限 iOS:
- `--packagemanager`: `SPM`、`Cocoapods`
