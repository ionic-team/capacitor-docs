# 构建 Ionic 照片库应用

**平台**：Web、iOS、Android

Capacitor 使构建可在 iOS、Android、桌面和 Web 上原生运行的 Web 应用变得简单。在本指南中，我们将构建一个可在所有平台上运行的完整照片库应用。

> 本指南现在由 Ionic 文档维护。[最新版本](https://ionicframework.com/docs/intro/next)包含 Angular、React 和 Vue 的教程。
