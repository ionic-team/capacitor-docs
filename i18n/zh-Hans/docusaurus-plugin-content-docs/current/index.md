# Capacitor：Web 应用的跨平台原生运行时

Capacitor 是一个跨平台原生运行时，它可以轻松构建在 iOS、Android 等平台上原生运行的高性能移动应用程序，并使用现代 Web 工具链。作为混合应用演进的下一步，Capacitor 创造了 **Web Native 应用**，为希望以 Web 为优先的团队提供现代原生容器方法，同时在需要时保留对原生 SDK 的完整访问权限。

![Capacitor Logo 和背景](/img/v6/docs/capacitor-index.png)

## 简介

Capacitor 提供了一组以 Web 为中心的一致 API，使应用尽可能接近 Web 标准，同时在支持它们的平台上访问丰富的原生设备功能。如果它在浏览器中工作，那么在使用 Capacitor 的移动应用中很可能也能工作。通过 iOS 上的 Swift、Android 上的 Java 和 Web 的 JavaScript 的插件 API，添加原生功能变得非常简单。

## 快速开始

开始使用 Capacitor 非常简单！Capacitor 可以集成到任何现有的现代 JavaScript 项目中，也可以从头开始创建全新的 Capacitor 项目。

按照 [安装指南](/main/getting-started/installation.md) 开始构建您的应用。
