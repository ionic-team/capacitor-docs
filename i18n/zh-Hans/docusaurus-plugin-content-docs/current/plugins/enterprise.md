# Capacitor 企业版

Capacitor 为关键任务企业应用提供强大的原生安全性、身份验证和数据管理解决方案及插件。

此外，团队还可以获得客户喜爱的支持和可选的咨询服务，以确保您的下一个主要应用项目取得成功并保持成功，无论移动生态系统如何变化。

[Capacitor 的企业产品](/plugins/enterprise.md)围绕一套强大的原生 Capacitor 插件展开: [Auth Connect](https://ionicframework.com/enterprise/auth-connect) 用于安全身份验证，[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 用于加密会话和用户身份管理，以及 [Offline Storage](https://ionicframework.com/enterprise/offline-storage) 用于高性能、加密的 SQL 数据存储。

每个解决方案都可以在 Capacitor 中运行的任何现代 Web 技术栈上良好运行，并且不需要 Ionic Framework。

## 企业身份验证

使用 [Auth Connect](https://ionicframework.com/enterprise/auth-connect) 和 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，使用移动安全最佳实践(例如生物识别身份验证和加密会话存储)连接到任何 OAuth 或 OpenID 提供商(Auth0、Ping、AWS Cognito、Azure Active Directory 等)。

Auth Connect 为身份验证提供了一个简单、可移植的 API，支持许多企业和消费者身份提供商，只需几行 JavaScript 代码即可提供复杂的原生身份验证流程。

结合 Identity Vault(如下所述)，Capacitor 应用可以实现最强大、最安全、最_正确_的企业身份验证流程。

## 安全身份管理

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 与 Auth Connect 协作，以保护敏感用户和身份数据(例如身份验证令牌)，并支持生物识别身份验证，支持最广泛的 iOS 和 Android 设备。Identity Vault 超越了典型的指纹或面部扫描插件，可与每个平台的安全飞地和加密服务一起使用。这是必要的，因为像大多数社区插件那样简单地扫描指纹或面部身份并不能提供足够的安全性和防止越狱或其他漏洞的保护。

[Identity Vault](https://ionicframework.com/enterprise/identity-vault) 也可以独立使用并与任何现有的身份验证流程集成，以提供高级会话安全性。

## 离线和加密数据存储

使用 [Offline Storage](https://ionicframework.com/enterprise/offline-storage)，Capacitor 开发人员可以访问高性能、加密的 SQL 数据存储系统，用于在 iOS 和 Android 上构建强大和安全的启用离线的应用。

结合 [Identity Vault](https://ionicframework.com/enterprise/identity-vault)，开发人员可以安全地管理数据加密密钥并实现设备上数据管理的最佳实践，而无需接触任何原生代码。
